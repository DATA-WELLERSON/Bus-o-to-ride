# Planos

Esta pasta guarda todos os planos de implementação do projeto **Busão to ride**.

## Convenção

- Cada plano é um arquivo Markdown próprio.
- Nome do arquivo: `AAAA-MM-DD-titulo-curto.md` (ex.: `2026-06-20-tela-resultados.md`).
- Estrutura sugerida de cada plano:

```markdown
# Título do plano

**Data:** AAAA-MM-DD
**Status:** rascunho | em andamento | concluído

## Objetivo
O que queremos alcançar.

## Passos
1. ...
2. ...

## Arquivos afetados
- `arquivo.html` — o que muda

## Observações
Riscos, dúvidas, decisões pendentes.
```
